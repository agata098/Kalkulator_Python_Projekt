# Kalkulator w języku Python
## Użyta biblioteka: Pygame
Projekt przedstawia model kalkuatora prostego z podświetlanymi przyciskami, gdy się najedzie na nie kursorem. Kalkulator przewiduje błąd dzielenia przez zero, a także kolejność wykonywania działań. Aby poprawnie uruchomić program, należy wcześniej zainstalować wymaganą bibliotekę za pomocą komendy "pip install pygame".
